# Extension-pdfbox

Uses pdfbox to merge PDF files together.

Each PDF doc is used as a layer and merged together.

# Maven dependency

```xml
<dependency>
  <groupId>com.github.codemonstur</groupId>
  <artifactId>extension-pdfbox</artifactId>
  <version>1.0.0</version>
</dependency>
```
